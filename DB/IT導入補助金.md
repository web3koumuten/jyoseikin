# デジタル化・AI導入補助金（旧：IT導入補助金）

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 補助金 |
| 管轄 | 中小企業庁（サービス等生産性向上IT導入支援事業） |
| 対象地区 | 全国 |
| 現在の状況 | 2026年度：「デジタル化・AI導入補助金2026」として実施 |
| 申請方式 | 電子申請（専用システム） |
| 提出先 | IT導入補助金申請システム（ITベンダーと共同申請） |
| 締切頻度 | 複数回（通年に近い形で締切あり） |
| 採択率目安 | 60〜80%程度（比較的通りやすい） |
| 申請〜入金までの期間 | 約6ヶ月〜1年 |

---

## 対象者

- **法人形態**：中小企業・小規模事業者・個人事業主
- **規模**：中小企業基本法に基づく中小企業
- **業種**：ほぼ全業種
- **除外**：大企業、ITベンダー・コンサル自身の導入

---

## 何に使えるか

ITツール・ソフトウェアの導入費用に使える。

**よく使われる用途**
- 会計・経理ソフト（freee・弥生等）
- 受発注・在庫管理システム
- 顧客管理（CRM）システム
- 労務管理・勤怠管理システム
- セキュリティソフト
- AIツール・自動化ツールの導入
- ECサイト構築ツール

**重要：登録された「IT導入支援事業者」が提供するツールのみ対象**

---

## 補助内容

| 項目 | 内容 |
|------|------|
| 補助率 | 1/2〜3/4（枠・ツールによって異なる） |
| 補助上限額 | 5万円〜450万円（枠によって異なる） |
| 主な枠 | 通常枠・インボイス枠・セキュリティ対策推進枠 |
| 自己負担 | 1/4〜1/2 |

---

## 診断ポイント（おすすめできる状況）

- 業務をデジタル化・効率化したい
- 会計・勤怠・顧客管理等のソフトを導入したい
- AIツールを業務に取り入れたい
- インボイス対応のためのシステムを入れたい
- セキュリティ対策を強化したい
- ITに不慣れな中小企業

---

## 診断ポイント（おすすめしにくい状況）

- 自社開発のシステムを作りたい（登録ツールのみが対象）
- ハードウェア単体の購入（原則ソフトウェアが主体）
- 大企業
- 導入後の活用イメージがない

---

## 必要書類

- gBizIDプライムの取得（必須）
- SECURITY ACTIONの宣言（必須）
- 直近の確定申告書または決算書
- ※ITベンダーと一緒に申請するため、ベンダー選定が先決

---

## よくある不採択理由

- gBizIDプライムの取得が間に合わない
- 登録されていないITツールを選んでいる
- 申請内容とツールの用途が合っていない
- 必要書類の不備

---

## 申請の流れ

1. gBizIDプライムを取得
2. SECURITY ACTIONの宣言
3. 登録済みのIT導入支援事業者（ITベンダー）を選定
4. ITベンダーと共同で電子申請
5. 審査・採択発表
6. ツール導入・支払い
7. 実績報告→補助金受取

---

## 注意事項

- **ITベンダーが登録事業者でないと申請できない**（ベンダー選定が最重要）
- 事業者単独では申請不可（必ずITベンダーと共同申請）
- ツール導入後の活用報告が必要
- gBizIDプライムの取得に2〜3週間かかる場合あり（早めに準備）
