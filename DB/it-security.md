# デジタル化・AI導入補助金 セキュリティ対策推進枠

## 概要
中小企業のサイバーセキュリティ対策強化のため、登録セキュリティサービス（UTM・EDR・セキュリティ監視サービス等）の導入費用の1/2（上限100万円）を補助する制度。不正アクセス・ランサムウェア被害が増加する中、製造業・医療・小売など幅広い業種でサイバー対策が急務となっています。IT導入補助金の一枠で、gBizIDプライムがあれば比較的シンプルに申請できます。

## 最大受給額
150万円（対象費用の1/2（補助額5〜150万円）。2年間のサービス利用料が対象。デジタル化・AI導入補助金2026の一枠）

## 受給条件
- サイバーセキュリティお助け隊サービスリスト登録サービスを導入すること
- gBizIDプライムアカウントの取得
- SECURITY ACTIONの宣言

## 対象企業規模
中小企業、小規模事業者

## 同時申請できる制度
it-donyu、hatarakikata-jikangai
