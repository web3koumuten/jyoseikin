# ものづくり・商業・サービス生産性向上促進補助金（ものづくり補助金）

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 補助金 |
| 管轄 | 中小企業庁（全国中小企業団体中央会） |
| 対象地区 | 全国 |
| 現在の回 | 第23次締切（2026年） |
| 申請方式 | 電子申請（専用システム） |
| 提出先 | ものづくり補助金電子申請システム |
| 締切頻度 | 年4〜6回程度 |
| 採択率目安 | 40〜50%程度（競争率が高い） |
| 申請〜入金までの期間 | 約1〜1.5年（採択→交付決定→事業実施→実績報告→入金） |

---

## 対象者

- **法人形態**：中小企業・小規模事業者・特定事業者
- **規模**：中小企業基本法に基づく中小企業（製造業は資本金3億円以下または従業員300名以下等）
- **業種**：製造業・サービス業・小売業等ほぼ全業種
- **除外**：大企業、発行済株式の総数・出資の総額の1/2以上を大企業が持つ企業

---

## 何に使えるか

革新的な製品・サービス開発や、生産プロセス・サービス提供方法の改善に使える。

**よく使われる用途**
- 機械装置・システム構築費（設備投資）
- ソフトウェア購入・導入費
- 専門家への外注・委託費
- クラウドサービス利用費
- 新製品・新サービス開発のための設備
- 生産ラインの自動化・効率化

**「革新性」が必須**（既存設備の単純な入れ替えは対象外）

---

## 補助内容

| 項目 | 内容 |
|------|------|
| 補助率 | 1/2（小規模事業者・再生事業者等は2/3） |
| 補助上限額（省力化投資促進枠） | 1,500万円〜5,000万円 |
| 補助上限額（製品・サービス高付加価値化枠） | 750万円〜1,250万円 |
| 自己負担 | 1/2以上 |

---

## 診断ポイント（おすすめできる状況）

- 設備投資で生産性を上げたい
- 新製品・新サービスを開発したい
- 省力化・自動化で人手不足を解消したい
- ある程度まとまった金額（数百万〜数千万）が必要
- 「革新的」な取り組みを説明できる

---

## 診断ポイント（おすすめしにくい状況）

- 単純な設備の入れ替え・修繕
- 革新性のない既存業務の継続
- 大企業（規模オーバー）
- すぐにお金が必要（入金まで1年以上かかる）
- 事業計画の策定が難しい（採択競争率が高い）

---

## 必要書類

- 事業計画書（独自フォーマットで作成、電子申請システムに入力）
- 決算書（直近2期分）
- 賃金引上げ計画書
- 法人：登記事項証明書
- 個人：確定申告書
- ※加点項目によって追加書類あり（経営革新計画等）

---

## よくある不採択理由

- 革新性・新規性が乏しい（「新しい取り組み」と言えない）
- 事業計画の数値根拠が弱い
- 補助対象経費の妥当性が不明確
- 市場性・成長性の説明不足
- 賃上げ計画が不明確

---

## 申請の流れ

1. 事業計画書の作成（専門家（認定支援機関）の確認が必要な場合あり）
2. 電子申請システムから申請
3. 審査・採択発表
4. 交付申請・交付決定
5. 補助事業の実施
6. 実績報告→確定検査→補助金受取

---

## 注意事項

- 採択されても「交付決定」前に発注・購入した経費は対象外
- 認定支援機関（税理士・商工会等）の確認書が必要な場合あり
- 賃上げ要件あり（給与支給総額・事業場内最低賃金の引上げ）
- 補助金は後払い（先に自分で支払い、後から補助金を受取）
