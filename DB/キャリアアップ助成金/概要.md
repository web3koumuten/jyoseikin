# キャリアアップ助成金

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 助成金（厚生労働省） |
| 管轄 | 厚生労働省・都道府県労働局・ハローワーク |
| 対象地区 | 全国 |
| 現在の状況 | 随時受付（締切なし） |
| 申請方式 | 書類申請（ハローワーク窓口へ持参または郵送） |
| 提出先 | 管轄のハローワーク（労働局） |
| 締切頻度 | 随時（取り組み実施後に申請） |
| 採択率目安 | 要件を満たせばほぼ支給（審査ではなく要件確認） |
| 申請〜入金までの期間 | 取り組み実施後の申請から約6ヶ月〜1年 |

---

## 対象者

- **法人形態**：法人・個人事業主
- **規模**：雇用保険適用事業所（規模問わず）
- **業種**：ほぼ全業種
- **除外**：雇用保険未加入事業所、支給申請日に雇用保険適用事業所でない場合

---

## 何に使えるか

非正規労働者のキャリアアップ・処遇改善の取り組みに対して支給される。

**主なコース・用途**
- **正社員化コース**：有期雇用→正社員転換
- **賃金規定等改定コース**：非正規の賃金を一定以上引き上げる
- **賃金規定等共通化コース**：正規・非正規の賃金規定を共通化する
- **賞与・退職金制度導入コース**：非正規に賞与・退職金制度を導入
- **社会保険適用時処遇改善コース**：社会保険適用拡大に伴う処遇改善
- **短時間労働者労働時間延長コース**：週所定労働時間を延長し社会保険を適用

---

## 補助内容（正社員化コースの例）

| 項目 | 内容 |
|------|------|
| 有期雇用→正社員転換 | 1人あたり80万円（大企業60万円） |
| 有期雇用→無期雇用転換 | 1人あたり40万円（大企業30万円） |
| 無期雇用→正社員転換 | 1人あたり40万円（大企業30万円） |
| ※賃金引上げ要件あり | 転換後6ヶ月の賃金が転換前比3%以上引上げ |

---

## 診断ポイント（おすすめできる状況）

- パート・アルバイト・契約社員を雇っている
- 非正規を正社員にする予定・検討している
- 非正規の賃金を引き上げたい
- 賞与・退職金制度を整備したい
- 雇用保険に加入している事業所
- 人材の定着・確保に課題がある

---

## 診断ポイント（おすすめしにくい状況）

- 従業員を雇用していない（一人社長等）
- 雇用保険未加入
- 正規・非正規の区別がない
- 転換・処遇改善の予定が全くない
- 過去に不正受給があった事業所

---

## 必要書類

- キャリアアップ計画書（様式第1号）※事前にハローワークに提出が必要
- 支給申請書（様式第3号）
- 雇用契約書・労働条件通知書
- 賃金台帳・出勤簿
- 転換・処遇改善を証明する書類
- ※コースによって必要書類が異なる

---

## よくある不支給理由

- キャリアアップ計画書を事前に提出していない（これが最多）
- 転換後の賃金引上げが3%未満
- 雇用期間の要件を満たしていない（6ヶ月以上の雇用等）
- 申請期限を過ぎた（取り組み後6ヶ月以内に申請等）
- 書類の不備・記載ミス

---

## 申請の流れ

1. **キャリアアップ計画書をハローワークに事前提出**（取り組み前に必須）
2. 計画に基づき取り組みを実施（正社員転換等）
3. 取り組み後6ヶ月分の賃金を支払い
4. 支給申請書・必要書類をハローワークに提出
5. 審査（約6ヶ月）
6. 助成金受取

---

## 注意事項

- **取り組み前にキャリアアップ計画書の提出が必須**（これを忘れると申請不可）
- 社労士が代行申請するケースが多い領域（報酬をもらっての代行は社労士法に注意）
- 毎年度、支給額・要件が改定される（最新情報を必ず確認）
- 不正受給は返還+罰則あり
