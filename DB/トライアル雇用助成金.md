# トライアル雇用助成金

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 助成金（雇用保険二事業） |
| 管轄 | 厚生労働省／ハローワーク |
| 申請方式 | 書類申請（管轄ハローワークへ） |
| 助成額目安 | 月4万円×3ヶ月（最大12万円/人）、障害者は月8万円×3ヶ月（最大24万円） |
| 採択率目安 | 要件を満たせば原則支給（審査型ではない） |
| 申請〜入金期間 | トライアル期間終了後の支給申請から約2〜4ヶ月 |
| 対象年度 | 令和8年度（2026年度） |
| 公式URL | https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/koyou_roudou/koyou/kyufukin/trial_koyou.html |

---

## 対象者（事業主側の要件）

- 雇用保険の適用事業主
- ハローワークの紹介状を受けて対象者をトライアル雇用した事業主
- トライアル期間（原則3ヶ月）中に雇用保険に加入していること
- トライアル終了後に本採用するかどうかを判断する意思があること（強制ではない）

---

## 対象となる労働者

就職が困難な状況にある求職者が対象。主な対象属性：

| 対象者 | 条件の目安 |
|-------|-----------|
| フリーター・長期失業者 | ハローワーク求職者で就職困難と認められた方 |
| 35歳未満のニート・無業者 | 就労経験の少ない若年層 |
| 障害者（精神・発達障害等） | ハローワーク等の紹介による |
| 就職氷河期世代 | 35〜55歳の不安定就労・長期無業者 |
| 子育て中の女性 | 育児のために離職していた方 |

---

## 何に使えるか

- 採用の見極め期間中（トライアル期間）の人件費負担の軽減
- 採用リスクを抑えながら多様な人材を試用できる制度
- ハローワーク経由の採用活動における費用補助

---

## 補助内容

### 一般トライアルコース

| 項目 | 内容 |
|------|------|
| 助成額 | 月4万円/人 |
| トライアル期間 | 原則3ヶ月 |
| 最大助成額 | 12万円/人 |
| 対象者数の上限 | 同時に雇い入れる人数に応じて一定上限あり |

### 障害者トライアルコース

| 項目 | 内容 |
|------|------|
| 助成額（精神・発達障害者等） | 月8万円/人 |
| 助成額（上記以外の障害者） | 月4万円/人 |
| トライアル期間 | 最大3ヶ月（障害の種類により延長可） |
| 最大助成額 | 24万円/人（精神・発達障害者） |

### 障害者短時間トライアルコース

| 項目 | 内容 |
|------|------|
| 助成額 | 月2万円/人 |
| 雇用時間 | 週10〜20時間未満 |
| 期間 | 3〜12ヶ月 |
| 延長 | 週20時間以上に移行後さらに助成あり |

## 診断ポイント（おすすめできる状況）

- 採用したい人材がいるが、スキルや適性に不安がある
- 新卒・第二新卒・フリーターの採用に積極的
- 精神障害者・発達障害者の採用を検討しているが、受け入れ体制に不安がある
- 採用コスト・リスクを下げながら人材確保したい
- ハローワーク経由の求人活動を行っている
- 就職困難者を積極的に採用したい（社会貢献・CSRの観点）

---

## 診断ポイント（おすすめしにくい状況）

- ハローワーク・認定機関以外のルート（求人サイト・縁故等）で採用した
- すでに本採用前提で雇用した（試用目的でない）
- 雇用期間が1ヶ月以内の短期・スポット雇用
- 対象属性の求職者を採用する予定がない
- トライアル期間後の本採用を全く想定していない（ただし本採用義務なし）

---

## 必要書類

### トライアル開始前
- トライアル雇用実施計画書（ハローワークへ）
- ハローワークの紹介状（求職者への発行分）
- 雇用契約書（トライアル期間を明記したもの）

### 支給申請時（トライアル期間終了後）
- トライアル雇用結果報告書兼支給申請書
- 賃金台帳（トライアル期間分）
- 出勤簿またはタイムカード
- 雇用保険被保険者証（写し）

---

## よくある不支給理由

- 事前にハローワークへトライアル実施計画書を提出していなかった
- 紹介状なしに雇い入れた（ハローワーク経由要件）
- トライアル期間中に解雇した（病気以外の事由）
- 賃金台帳・出勤簿が整備されていなかった
- 申請期限（トライアル期間終了後1ヶ月以内）を過ぎた

---

## 申請の流れ

1. **ハローワークへの求人登録**（トライアル雇用希望の旨を明示）
2. **ハローワークから紹介状の発行**（対象の求職者に発行）
3. **トライアル雇用実施計画書の提出**（ハローワークへ。雇入れ前または当日まで）
4. **トライアル雇用開始**（原則3ヶ月、雇用保険加入）
5. **トライアル期間中の記録整備**（出勤簿・賃金台帳）
6. **結果報告・支給申請**（トライアル終了後1ヶ月以内）
7. **審査・支給決定**（申請から約2〜4ヶ月）

---

## 注意事項

- トライアル終了後の**本採用は義務ではない**（合わなければ終了でもOK）
- ただし、最初から本採用しない前提でのトライアル利用は趣旨に反するため、誠実な採用活動が前提
- トライアル期間中の解雇は、本人の重大な問題（犯罪等）以外は返還事由になり得る
- 特定求職者雇用開発助成金との組み合わせが可能（トライアル→本採用後に申請）
- 障害者のトライアルは精神・発達障害者の受け入れに特に活用されており、社会的ニーズが高い
