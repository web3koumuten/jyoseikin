# 両立支援等助成金

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 助成金（雇用保険二事業） |
| 管轄 | 厚生労働省／各都道府県労働局雇用環境・均等部（室） |
| 申請方式 | 書類申請（都道府県労働局へ郵送または持参） |
| 助成額目安 | コースにより異なる（20万円〜110万円/人 等） |
| 採択率目安 | 要件を満たせば原則支給（審査型ではない） |
| 申請〜入金期間 | 申請から約3〜6ヶ月 |
| 対象年度 | 令和8年度（2026年度） |
| 公式URL | https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kodomo/shokuba_kosodate/ryouritsu01/index.html |

---

## 対象者

- 雇用保険の適用事業主
- 育児・介護と仕事の両立支援に取り組む事業主
- 各コースの要件（制度の整備・対象者の要件等）を満たしていること

---

## 何に使えるか

従業員が育児・介護をしながら働き続けられる職場環境の整備に取り組む際の費用補助。

主なコース：

| コース名 | 主な内容 |
|---------|---------|
| 出生時両立支援コース（子育てパパ支援助成金） | 男性労働者が育児休業・育児目的休暇を取得した場合 |
| 介護離職防止支援コース | 介護休業取得・復帰支援、介護対応のための職場環境整備 |
| 育児休業等支援コース | 育休取得・復帰支援の制度整備と実施 |
| 育児休業等に関する情報公表支援コース | 育休取得実績等の情報公表 |
| 不妊治療両立支援コース | 不妊治療と仕事を両立できる制度の整備・利用 |

---

## 補助内容

### 出生時両立支援コース（子育てパパ支援助成金）

男性労働者が育児休業または育児目的休暇を取得した場合の助成。

| 項目 | 助成額（中小企業） |
|------|----------------|
| 育休取得（1人目） | 20万円 |
| 育休取得（2人目以降） | 10万円 |
| 育休取得率等の目標達成 | 追加助成あり |
| 業務体制整備（代替要員確保等） | 20万円加算 |

### 介護離職防止支援コース

| 項目 | 助成額（中小企業） |
|------|----------------|
| 雇用環境整備（制度整備・周知等） | 100万円（初回） |
| 介護休業取得・職場復帰 | 30万円/人 |
| 介護のための柔軟な就労形態制度整備 | 60万円（新規） |

### 育児休業等支援コース

| 項目 | 助成額（中小企業） |
|------|----------------|
| 育休取得時（育休復帰支援プラン策定） | 60万円/人（取得＋復帰） |
| 代替要員確保時 | 60万円加算 |
| 有期契約労働者の育休取得 | 60万円/人 |

### 不妊治療両立支援コース

| 項目 | 助成額（中小企業） |
|------|----------------|
| 環境整備（制度整備） | 30万円 |
| 不妊治療のための休暇等取得 | 30万円/人（最大2名） |

---

## 診断ポイント（おすすめできる状況）

- 従業員から育児休業・介護休業の相談が来ている
- 男性社員の育休取得率を上げたい（社会的トレンド・採用力強化）
- 介護を理由とした退職者が出ている、または出そう
- 子育て中の女性従業員の定着・復帰支援を強化したい
- 不妊治療中の従業員のサポート体制を整えたい
- 従業員の採用・定着に課題がある（両立支援整備でアピールしたい）
- 「子育てサポート企業」（くるみん認定）等の取得を目指している

---

## 診断ポイント（おすすめしにくい状況）

- 雇用保険の適用事業所でない
- 育児・介護に関する制度（休業規定等）を就業規則に定める意思がない
- 対象となる育休・介護休業取得者がいない、かつ取得予定もない
- 過去に不正受給があった事業主

---

## 必要書類

### コース共通（環境整備段階）
- 就業規則または育児・介護休業規程（両立支援に関する規定を整備したもの）
- 労働者への周知証明（社内通知文書等）

### 出生時両立支援コース
- 育児休業取得確認書類（育休期間・取得日）
- 対象労働者の賃金台帳・雇用契約書
- 業務体制整備の証明書類（代替要員の雇用契約書等）

### 介護離職防止支援コース
- 介護支援プラン（個別に策定）
- 介護休業取得・復帰証明書類

### 育児休業等支援コース
- 育休復帰支援プラン
- 育休取得・職場復帰証明書類
- 代替要員の雇用契約書（加算申請の場合）

---

## よくある不支給理由

- 就業規則に両立支援制度が明記されていなかった
- 育休取得前に支援プランを策定していなかった
- 申請期限（復帰から2ヶ月以内等）を過ぎた
- 対象労働者の育休期間が規定の日数に満たなかった
- 代替要員が「新規雇用」でなく内部異動だった（要件を確認）

---

## 申請の流れ

1. **制度整備**（就業規則への両立支援規定の追加）
2. **従業員への周知**（制度の社内通知）
3. **対象者への支援プラン策定**（育休・介護休業取得前に）
4. **育休・介護休業の取得**
5. **職場復帰**（育休・介護休業取得後）
6. **支給申請**（復帰から一定期間後）
7. **審査・支給決定**

---

## 注意事項

- 各コースで**事前の制度整備・計画策定が必須**（取得後に整備してもNG）
- 育休復帰支援プランは対象者との面談のうえ策定する必要あり
- 男性育休のコースは取得日数・期間の要件があるため注意（例：連続5日以上等）
- 同一労働者への複数コース申請は基本不可（内容が重複する場合）
- 令和6年度に育児・介護休業法が改正されているため、最新の法令・規定に合わせた就業規則の整備が必要
