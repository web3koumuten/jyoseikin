# 事業再構築補助金

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 補助金 |
| 管轄 | 中小企業庁 |
| 対象地区 | 全国 |
| 現在の状況 | ⚠️ **2024年終了（第12回が最終回）** |
| 申請方式 | 電子申請（Jグランツ）※終了 |
| 締切頻度 | 終了のため新規申請不可 |
| 採択率目安 | 最終回（第12回）：約30〜40% |
| 申請〜入金までの期間 | 採択から約1〜2年 |

---

## ⚠️ 重要：この補助金は2024年に終了しています

新規申請は受け付けていません。採択済みの事業者のみ事業実施中。

---

## 対象者（過去の制度）

- **法人形態**：中小企業・中堅企業・個人事業主・企業組合等
- **規模**：中小企業〜中堅企業（売上高制限あり）
- **条件**：コロナ等の影響で売上が減少、新分野展開・業態転換等を行う
- **業種**：ほぼ全業種

---

## 何に使えるか（過去の制度）

思い切った事業再構築（新分野展開・業態転換・事業転換等）に使えた。

**よく使われた用途**
- 全く新しいビジネスへの参入（設備投資・改装等）
- 既存店舗の業態転換（例：飲食→テイクアウト専門店）
- EC・オンライン事業への転換
- 新製品・新サービスの開発

---

## 補助内容（過去の制度・参考）

| 項目 | 内容 |
|------|------|
| 補助率 | 1/2〜2/3 |
| 補助上限額 | 100万円〜3億円（枠・規模によって異なる） |
| 主な枠 | 成長枠・グリーン成長枠・産業構造転換枠等 |

---

## 診断ポイント

**現在は新規申請不可のため、この補助金を提案することはできない。**

類似の用途であれば以下を代替として検討：
- 設備投資 → **ものづくり補助金**
- 販路開拓 → **小規模事業者持続化補助金**
- IT導入 → **IT導入補助金（デジタル化・AI導入補助金）**

---

## 注意事項

- 新規申請は一切受け付けていない
- 後継・類似制度の情報は随時確認が必要
- 過去の採択事業者は引き続き事業実施・実績報告の義務あり
