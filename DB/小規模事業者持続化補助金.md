# 小規模事業者持続化補助金（一般型・通常枠）

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 補助金 |
| 管轄 | 中小企業庁（商工会議所・商工会経由） |
| 対象地区 | 全国 |
| 現在の回 | 第19回（2026年） |
| 申請方式 | 電子申請（Jグランツ） |
| 提出先 | Jグランツ経由、商工会議所が支援 |
| 締切頻度 | 年3〜4回程度 |
| 採択率目安 | 60〜70%（比較的通りやすい） |
| 申請〜入金までの期間 | 早くて約6ヶ月〜1年（採択後に事業実施→実績報告→入金） |

---

## 対象者

- **法人形態**：法人・個人事業主・特定非営利活動法人（NPO）
- **規模**：小規模事業者（商業・サービス業は従業員5名以下、製造業等は20名以下）
- **業種**：ほぼ全業種（一部除外あり）
- **除外**：大企業の子会社、医師・弁護士等の士業、風俗営業等

---

## 何に使えるか

販路開拓・生産性向上のための取り組みに使える。

**よく使われる用途**
- チラシ・パンフレット作成
- ホームページ制作・リニューアル
- Web広告（Google・Meta等）※上限あり
- 展示会・見本市への出展
- 店舗改装・設備導入
- 新商品・新サービスの開発

**注意：ウェブサイト関連費のみの申請は不可**（他の経費と組み合わせる必要あり）

---

## 補助内容

| 項目 | 内容 |
|------|------|
| 補助率 | 2/3 |
| 補助上限額（通常枠） | 50万円 |
| インボイス特例加算 | +50万円（条件あり） |
| 自己負担 | 1/3 |

---

## 診断ポイント（おすすめできる状況）

- 新規顧客を増やしたい
- 販促物・ホームページを整備したい
- 展示会に出たい
- 設備を導入して業務を改善したい
- 従業員5名以下（製造業は20名以下）
- 個人事業主・小規模法人

---

## 診断ポイント（おすすめしにくい状況）

- 大企業・中堅企業（規模オーバー）
- 人件費・給与を補助金で賄いたい
- すでに同補助金で採択済み（同年度内）
- 士業・風俗営業
- すぐにお金が必要（入金まで半年〜1年かかる）

---

## 必要書類

- 経営計画書・補助事業計画書（Jグランツ上で入力）
- 事業支援計画書（商工会議所が発行・様式4）
- 確定申告書（直近1期分）
- 法人：履歴事項全部証明書
- ※特例・加点を希望する場合は追加書類あり

---

## よくある不採択理由

- 事業計画の具体性が低い（「何をするか」が曖昧）
- 補助対象経費がウェブサイト関連費のみ
- 小規模事業者の要件を満たしていない
- 販路開拓との関連性が薄い
- 数値目標が設定されていない

---

## 申請の流れ

1. 商工会議所（または商工会）に相談・事業支援計画書（様式4）を依頼
2. Jグランツで電子申請
3. 審査・採択発表
4. 補助事業の実施
5. 実績報告→確定検査→補助金受取

---

## 注意事項

- 採択後に事業を実施（先払い→後から補助金受取）
- 事業完了後に実績報告が必要
- 商工会議所への相談が申請前に必須（様式4の発行に時間がかかる場合あり）
- 締切は年数回のため、次回を逃すと数ヶ月待ちになる
