# 雇用調整助成金

## 基本情報

| 項目 | 内容 |
|------|------|
| 種別 | 助成金（厚生労働省） |
| 管轄 | 厚生労働省・都道府県労働局・ハローワーク |
| 対象地区 | 全国 |
| 現在の状況 | 随時受付（通常制度として継続中） |
| 申請方式 | 書類申請（ハローワーク窓口へ持参または郵送） |
| 提出先 | 管轄のハローワーク（労働局） |
| 締切頻度 | 随時（休業実施後に申請） |
| 採択率目安 | 要件を満たせばほぼ支給（審査ではなく要件確認） |
| 申請〜入金までの期間 | 休業実施の翌月以降に申請、支給まで約2〜6ヶ月 |

---

## 対象者

- **法人形態**：法人・個人事業主
- **規模**：雇用保険適用事業所（規模問わず）
- **業種**：ほぼ全業種
- **条件**：売上・生産量が最近3ヶ月間で10%以上減少していること
- **除外**：雇用保険未加入事業所

---

## 何に使えるか

景気悪化・売上減少時に従業員を休業・教育訓練させることで、雇用を維持するための助成金。

**具体的な使い方**
- 売上が落ちて従業員を休ませざるを得ない時
- 休業手当を支払う原資が足りない時
- リストラせずに雇用を守りたい時
- 従業員を休業中に研修・教育訓練させる時

---

## 補助内容

| 項目 | 内容 |
|------|------|
| 助成率（中小企業） | 休業手当の2/3（教育訓練の場合は+1,200円/人日） |
| 助成率（大企業） | 休業手当の1/2 |
| 1人1日あたりの上限 | 8,490円（2024年度） |
| 支給限度日数 | 1年間で100日（教育訓練は別途） |

---

## 診断ポイント（おすすめできる状況）

- 売上・生産量が3ヶ月で10%以上落ちている
- 従業員を休業させる必要がある
- リストラ・解雇を避けたい
- 雇用保険に加入している事業所
- 一時的な経営悪化で将来的に回復見込みがある

---

## 診断ポイント（おすすめしにくい状況）

- 売上減少が10%未満（要件を満たさない）
- 従業員を雇用していない（一人社長等）
- 雇用保険未加入
- 恒常的な経営不振（一時的でない）
- 現在は景況が回復しており休業の必要がない（コロナ禍と比べ現在は申請件数が激減）

---

## 必要書類

- 休業等実施計画届（事前に提出が必要）
- 支給申請書
- 賃金台帳・出勤簿・休業日の記録
- 売上減少を示す書類（売上台帳等）
- 休業協定書（労働組合または労働者代表との協定）

---

## よくある不支給理由

- 休業等実施計画届を事前に提出していない
- 売上減少10%の要件を満たしていない
- 休業手当の支払いが適切でない（60%未満など）
- 書類の不備・記載ミス
- 申請期限を過ぎた

---

## 申請の流れ

1. **休業等実施計画届をハローワークに事前提出**
2. 従業員に休業手当を支払いながら休業を実施
3. 休業終了後に支給申請書・必要書類を提出
4. 審査（約2〜6ヶ月）
5. 助成金受取

---

## 注意事項

- コロナ禍（2020〜2022年）に特例で大幅拡充されたが、現在は通常制度に戻っている
- 現在の申請件数はコロナ禍と比べ大幅に減少（需要が低い状況）
- 社労士が代行申請するケースが多い（社労士法に注意）
- 毎年度、支給額・要件が改定される
- 計画届の事前提出を忘れると申請不可
