# 助成金・補助金 データベース

助成金診断サービス用の助成金・補助金情報データベースです。

## 構成

```
DB/
├── 小規模事業者持続化補助金.md
├── ものづくり補助金.md
├── IT導入補助金.md
├── 事業再構築補助金.md
├── 雇用調整助成金.md
├── 人材開発支援助成金.md
├── 業務改善助成金.md
├── 両立支援等助成金.md
├── 特定求職者雇用開発助成金.md
├── トライアル雇用助成金.md
└── キャリアアップ助成金/
    ├── 概要.md
    ├── 正社員化コース.md
    ├── 障害者正社員化コース.md
    ├── 賃金規定等改定コース.md
    ├── 賃金規定等共通化コース.md
    ├── 賞与・退職金制度導入コース.md
    └── 短時間労働者労働時間延長支援コース.md
```

## MDファイルの構成

各ファイルは以下のセクションで統一されています：

- 基本情報（種別・管轄・助成額・採択率・申請期間）
- 対象者
- 何に使えるか
- 補助内容
- 診断ポイント（おすすめできる状況）
- 診断ポイント（おすすめしにくい状況）
- 必要書類
- よくある不支給理由
- 申請の流れ
- 注意事項
- 更新履歴
